This directory contains the original SVG files used to generate PNG elements via a python script based on Pillow.
For mobile, they are not standalone and require theme-specific files called common.css and svg.css to be placed in the same directory.

A __init__.py file is not necessary here as the folders are *not* imported by the main program. They are used only in the development phase to generate PNGs.
